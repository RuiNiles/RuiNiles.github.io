Tree Set Data Structure

View Code Made: 06/10/20

This is a piece of university coursework where we were required to create a recursive set data structure using classes that inherit their methods from a single abstract class which acts as an interface to the user. The project also called for methods to add integers to the set, check if the set contains a specific integer, create the union of two sets and print the contents of a set.

The code is given as an intelliJ project file. However the class files can be added to any project as long as they are all included (They can be found in the 'src' folder). Once a set(IntSet) object is created you can use the add, contains, union, and toString methods to manipulate it.

Genetic Algorithms

View Code Made: 06/10/20

My first attempts at creating genetic algorithms to solve black box and knapsack problems. The project includes the class 'Assess' which contains an example 20 number blackbox algorithm and a set of 100 knapsack item values to test the two algorithms.

The black box problem is an algorithm that takes an input of 20 numbers and outputs 1 number calculated using the input. The exact nature of the algorithm is unknown so there isn't an efficient method to find the 20 numbers that give the highest/lowest output. Hence why a genetic algorithm is used to find it.

The knapsack problem is a set of "items" where each item has a weight and utility value. Each item can be taken or left. The aim is to find the combination of items that give the highest utility without exceeding the maximum weight.

To run the demo of the project download all class files into a single directory compile them all and run the example.java file. This project is designed to be run in the command line.
